🚀 About  
This isn’t a typical portfolio. It’s an interactive space environment where each section of my work exists as a planet you can explore. The starfield above isn’t random — it’s generated from real astronomical data and updates based on your location and time.

🛠️ Built With  
- Next.js + JavaScript  
- React Three Fiber (Three.js)  
- Tailwind CSS  
- Vercel (hosting & deployment)

✨ Features  
- Real-time starfield using astronomical coordinates (RA/Dec → Alt/Az)  
- Location-aware sky via geolocation + local sidereal time  
- Interactive 3D navigation between “planet” sections  
- Shader-based visuals with brightness mapped from star magnitude  
- Relativistic-inspired effects (redshift / blueshift on motion)  
- Animated UI system with modal “data nodes” and transmission panels
